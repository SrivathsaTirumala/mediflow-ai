## 🚀 DEPLOYMENT STEPS (Copy-Paste Ready)

### For Vercel CLI Users:
```bash
# Step 1: Install Vercel
npm install -g vercel

# Step 2: Create folder
mkdir mediflow-ai-demo
cd mediflow-ai-demo

# Step 3: Copy HTML file and rename
cp mediflow-ai-standalone.html index.html

# Step 4: Deploy
vercel

# Step 5: Get your URL!
# Your live demo: https://mediflow-ai-demo.vercel.app
```

### For GitHub + Vercel Users:
```bash
# Step 1: Create GitHub repo
# https://github.com/new → "mediflow-ai-demo"

# Step 2: Clone it
git clone https://github.com/YOUR-USERNAME/mediflow-ai-demo.git
cd mediflow-ai-demo

# Step 3: Add HTML file
cp mediflow-ai-standalone.html index.html
git add index.html
git commit -m "Add MediFlow AI demo"
git push origin main

# Step 4: Go to Vercel
# https://vercel.com/new → Select GitHub repo → Deploy

# Your live demo: https://mediflow-ai-demo.vercel.app
```
